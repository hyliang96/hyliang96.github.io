使用

创建demo.py

执行目标文件

python demo.py

打开TensorBoard

tensorboard --logdir runs --port=6006

默认端口6006，可更改

本地访问TensorBoard

tb 4 6006

  • 4是gpu服务器的编号
  • 6006是tensorboard占用的服务器端口，需保持和上面的一致
